This batch program is used to generate encrypted password for wizBank db configuration.

1. Edit 'encrypt.bat' file, set JAVA_HOME parameter.
2. Run Encrypt.bat.
3. Enter user's name and user's password step by step.
4. Copy generate encrypted password to wizBank db configuration.

eg:
----print start-----

	Please enter user name:
	username
	Please enter user password:
  123456
  The encrypted password is:40M+kPhe
  press any key to stop.
  
----print end-----  
  
  
  Copy '40M+kPhe' to db configuration.

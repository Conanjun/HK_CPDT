学习小组
首页学习小组gadget

学员在首页可以看到此学习小组gadget，这里只列出当前学员管理的或已加入的最新的5个学习小组，点击学习小组名称可链接到学习小组主页；点击“更多”可链接到学习小组列表。
学习小组列表
学员在菜单中点击学习小组，可以进入到以下页面：

说明：
    
  1. 每个列表分别显示该学员管理的、加入的、未加入的所有学习小组。 
  2. “广州小组01”——学习小组名称。 
  3. “X个成员”——学习小组内成员数（包含组长）。 
  4. “X条信息”——学习小组的讨论区内论题数。 
  5. “X个资源”——学习小组内资源数。
  1. 管理中的学习小组：
    1. 当前学员为组长的学习小组，组长拥有编辑小组资料、维护（添加\修改\删除）资源权限。
    2. 每个学习小组可以有多个组长，组长或成员退出小组由TA手动删除。
    3. 点击学习小组名称可链接到学习小组主页。
    4. 无管理的学习小组时隐藏此gadget（仅限管理中学习小组gadget）
  2. 已参加的学习小组：
    1. 当前学员已加入但不是组长的学习小组，组员拥有添加资料、删除\更改自己上传资料的权限。
    2. 点击学习小组名称可链接到学习小组主页。
  3. 其他公开的学习小组：
    1. 列出的学习小组为所有包含该学员用户组所在TC下的所有该学员未加入的已公开的学习小组。
    2. 学习小组名称为只读(非链接)，未加入该小组的学员无法查看其资料。
    3. “联系”：显示该小组所在培训中心的培训管理员的email（为链接，点击弹出写邮件窗口）。若有多个培训管理员，则根据名字排序显示第一个培训管理员email，若该培训中心无培训管理员或所有培训管理员都无email，则不显示email。
  4. 搜索学习小组：
    1. 通过关键字针对学习小组的名称进行模糊搜索，搜索结果显示于搜索框下方。
    2. 空搜索时给出提示：“请输入关键字！”。
    3. 搜索结果为该学员的用户组所在的TC下所有符合搜索条件的公开的学习小组。若无相匹配的数据，则显示“内容为空”。
    4. 搜索结果中，学员已加入的学习小组名为链接，点击进入学习小组主页。
    5. 搜索结果中，若有学员未加入的学习小组，在该学习小组后有“联系”email。

学员看到的学习小组主页与培训管理员的管理学习小组页面一样，只是权限不同而已，下面介绍成员、组长与培训管理员权限的区别：
  1. 学员：查看资源、添加\删除\更改自己添加的资源；查看成员资料；查看\发表讨论区内论题和文章。
  2. 组长：编辑小组资料；查看\添加\删除\更改所有资源；查看成员资料；查看\发表讨论区内论题和文章。
  3. 培训管理员：编辑小组资料；查看\添加\删除\更改所有资源；查看成员资料，添加\删除小组成员；查看\发表\删除讨论区内论题和文章。

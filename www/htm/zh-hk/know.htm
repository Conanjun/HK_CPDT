在线问答
系统中的所有学员都可以参与在线问答进行交流与分享。
  1. 在线问答主页
    学员点击菜单中在线问答，进入在线问答主页，如下图：
    1. 搜索

      1. 此搜索针对“在线问答”中的问题进行模糊搜索。
      2. 下拉框：默认选择全部分类。下拉框内的选项为所有一级分类，没有一级分类时只显示“全部分类”。
      3. 高级搜索：点击进入高级搜索页面，参考高级搜索-在线问答。
      4. 点击搜索，进入搜索结果页面，如下图：

        
      1. 分3个TAB分别显示已解决问题、待解决问题和精选问题中的搜索结果。（若通过高级搜索而来，则默认显示的TAB为高级搜索条件中选择的状态；选择多状态时，优先级为：待解决>已解决>精选；未选择的状态TAB中显示“内容为空”） 
      2. 在列表右上角有两个下拉框用于选择排序方式。第1个为选择排序类型下拉框，包含标题、提问时间、分类和评价人数（选择待解决问题TAB时，不显示“评价人数”）4个选项，默认选择“提问时间”；第2个为选择排序方式下拉框，有“升序”和“倒序”两个选项可选，默认选择“倒序”。 
      3. 问题的标题为链接，点击链接到查看问题页面。 
      4. 黄色背景的内容为问题标题，蓝色背景的内容为问题补充说明。 
      5. 分类和发问者为链接，点击分类进入该分类主页，点击发问者的昵称弹出窗口显示该用户公开信息。 
      6. 正面评价(选择待解决问题TAB时，不显示)：“100%”是根据投票“好”的票数占总票数的比例算出，即：正面评价=好/（好+不好）。（已解决和精选问题才有）
      7. 评价人数：显示当前已评价的人数。（已解决和精选问题才有） 
      8. 分类：显示该问题所在分类；发问时间：显示该问题的发问日期；发问者：显示发问者的昵称。 
      9. 若无结果则显示“内容为空”。
    2. 统计gadget

      此处显示已解决和待解决问题的统计数量。
       已解决问题数：显示系统内已解决问题的总数。
       待解决问题数：显示系统内待解决问题的总数。
    3. 问题分类gadget
        
      1. 显示所有一级分类（按包含的问题数倒序排序）和该分类目录下最受欢迎（即：问题数最多的）的三个子分类，若子分类多于三个则用“…”代替。 
      2. 顶层分类右侧括号内显示此分类下包含的问题总数（包含所有子分类中的问题数），总数为0时显示0。 
      3. 所有分类均为链接，点击分类名可以直接进入此分类主页。 
      4. 点击详细分类，进入详细分类页面，总览所有分类，如图：
        
      1. 灰色背景的为一级分类，一级分类下显示所有二级分类。 
      2. “内容为空”表示此一级分类下无二级分类。 
      3. 点击分类名可链接到所选分类主页，如下图：

      1. 问题分类gadget：显示此分类下的子分类，分类后括号内显示该分类下的问题总数。
      2. 点击问题分类gadget中的“返回上级”，返回到上级分类页面。
      3. 相关课程gadget：
        1. 显示的课程：根据该问题所在分类及父分类的分类名作为搜索关键字，与当前用户所能看到的培训（即：所有当前用户的用户组所在培训中心下的所有发布的课程/测试/参考资料）的所有信息进行匹配搜索得出（即：结果与高级搜索内针对培训的全文搜索结果一致）。
        2. 若无相关培训，则不显示此gadget。
        3. 以培训发布时间倒序排列。
        4. 只显示最新发布的5个培训。
    4. 问题列表gadget

       分4个TAB分别显示待解决问题、已解决问题、精选问题和FAQ。
       未选择分类时，列表中以问题的提问时间倒序排列显示所有问题。
      待解决问题：
      1. 显示状态为待解决的问题。
      2. 排序：默认按提问时间倒序排序。
      3. 该列表采用分页显示，默认为每页40个。
      4. 分类：显示此问题所在分类，点击分类名可链接进入该分类页面。
      5. 标题：显示问题的标题。点击问题标题进入查看问题页面。
      6. 回答数：显示目前该问题得到的回答总数，没有回答则显示0。
      7. 提问时间：显示该问题提出的时间。
      已解决的问题：
      1. 显示状态为已解决的问题。
      2. 其他与A. 待解决问题类似。
      精选问题：
      1. 精选问题：仅为从已解决问题中抽取的部分经典问题，不改变问题原来的任何属性（该问题变为精选问题后同时在已解决问题中显示）。当问题状态变为已解决问题后，在问题下方会出现投票区，在规定的可投票时间内，学员可根据自己的评价选择“好”或“不好”。系统根据每个评选精选问题周期内，投“好”与“不好”的票数来选取精选问题，详细计算方式见下文“计算方式”。
      2. 计算方式：在每个评选精选问题周期，系统会根据周期内投票结果(只取当前周期内的临时投票结果，不累积之前周期的投票结果，每个周期结束时，临时投票结果重置为0)中“好”-“不好”的票数差，取差值最大的（至少要大于0）的前100个问题（不足100个时，则有多少取多少）列入精选问题。
      3. 排序：以投“好”的票数（这里指“好”的历史总票数）倒序排列。
      FAQ：
      1. 所有问题都是培训管理员添加的，学员无法添加和回答FAQ问题，只有查看权限。
      2. 其他与待解决问题类似。

    帮助简单的介绍了在线问答中精选问题的评选规则。点击“帮助”按钮进入以下页面：
  3. 查看问题
    待解决问题

    1. 待解决问题gadget：
      1. 显示问题信息，包含：提问者图片、昵称、提问时间及回答数（此问题目前被回答次数）。
      2. “取消提问”：该问题提出者在未采纳最佳答案前（即：待解决问题）可点击此按钮删除该问题。
      3. 问题标题加粗显示，问题补充说明显示在标题下。
      4. “我来回答”：点击后跳至页面最下方我的回答gadget。
      5. “问题补充”：点击进入我要提问页面修改问题补充说明。这时，“我的提问”和“问题分类”内容不可修改。“确定”后，返回查看此问题页面，“取消”也返回到此。
      6. “采纳为答案”：点击后将该回答采纳为最佳答案，同时该问题变为已解决问题。（只能由提问者指定，可以指定自己的回答作为最佳答案，但提问者自己的回答被指定为最佳答案不加“被采纳为最佳答案”的20分）
    2. 回答gadget：
      1. “回答”后括号内显示回答总数。
      2. 回答者信息包含：回答者图片、回答者昵称、回答时间、回答内容及参考资料。
      3. 回答以时间升序排列，最早的回答在最上面。
    3. 相关问题gadget：
      1. 显示此问题所在分类下的与此问题同状态的最新的5个问题。（说明：此问题为待解决问题，所在目录：英语；则相关问题为英语目录下的最新的5个待解决问题，且只显示问题标题。）
      2. 问题以提问时间倒序排列。
      3. 点击问题标题进入所选问题查看页面。
      4. 若无相关问题则显示“内容为空”。
    4. 我来回答：（提问者也可以回答自己的问题，但自己回答不加分）
      1. 点击“我来回答”按钮，跳至页面最下方回答问题gadget。
      2. 我的回答：必填项，字数限制：10000。
      3. 参考资料：非必填项，字数限制：255。
      4. 确定：提交回答。
    5. 相关培训：
      1. 显示的课程：根据该问题所在分类及父分类的分类名作为搜索关键字，与当前用户所能看到的培训（即：所有当前用户的用户组所在培训中心下的所有发布的课程/测试/参考资料）的所有信息进行匹配搜索得出（即：结果与高级搜索内针对培训的全文搜索结果一致）。
      2. 以培训发布时间倒序排列。
      3. 只显示最新发布的5个培训。
    已解决问题

    1. 最佳回答：显示提问者指定的最佳回答。
    2. 投票区：
      1. “目前有0个人评价”，显示已投票的总人数。
      2. 好/不好按钮下方分别显示“50% （0）”，括号中0分别表示投好/不好的票数；50%分别表示投好/不好占总票数的比例。
      3. 任何学员都可以参与投票。每人只可以对同一问题投1次票，可以投好或不好，也可以不投。
      4. 若学员对当前问题已投过票，点击“好”或“不好”再次投票时，弹出窗口提示信息：“对不起！您不能重复评价”。
      5. 投票具有周期性，假设周期为1星期，即从问题变为已解决问题开始接受投票时算起，一周后关闭投票，将“您觉得最佳答案好不好”改为“评价已经被关闭”，“好”/“不好”按钮变灰（不可点击），从而结束投票。如下图：
    FAQ

    FAQ问题学员不能回答和投票，也不会进入精选问题，除此之外，其他都与已解决问题类似，可参考已解决问题。
  4. 我要提问
    学员提问入口，点击“我要提问”进入提问页面，如图：
    1. 我的提问：必填项。输入问题标题，限60个字。
    2. 问题补充说明：输入问题补充说明，非必填项，字数限制：500。
    3. 问题分类：必填项，左边选择框内为一级分类（必选，默认选择第一个分类），右边为二级分类（非必选）。当选择某个一级分类时，右边也随之刷新显示该一级分类下的所有二级分类，可以只选择一级分类而不选择二级分类（不可同时选择多个一级或二级分类）。（若培训管理员未创建任何分类，则此处显示为“内容为空”）
    4. 确定：对以上信息进行验证，若通过则成功提问，若不通过则给出相应提示。
    5. 取消：返回在线问答首页。

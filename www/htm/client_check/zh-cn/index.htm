安装Java插件
  1. 首先，检查是否已经安装Java插件：
    1. 打开浏览器的 工具 > Internet 选项 > 高级
    2. 检查 "Java (Sun)" 部分是否存在，如下图
    3. 如果 "Java (Sun)" 部分不存在，请根据下面步骤 b. 安装Java插件
    4. 如果 "Java (Sun)" 部分下面的选项没有如上图所示被选上，请选上该选项并重新启动浏览器。再次进入学习系统，检查是否需要更新Java插件
  2. 如果没有Java插件，请根据以下步骤安装：
    （请注意，您必须拥有操作系统的管理员权限才能进行安装）
    1. 下载Java插件(15.2MB)
    2. 安装Java插件
    3. 重新启动浏览器并进入学习系统

再次检查

更新Java插件
如果使用的是旧版本的Java插件，请根据以下步骤更新：
（请注意，您必须拥有操作系统的管理员权限才能进行安装）
  1. 下载新的Java插件(15.2MB)
  2. 打开 控制面板 > 添加/删除程序，删除程序 "Java 2 Runtime Environment" 或者 "J2SE Runtime Environment"
  3. 安装新的Java插件
  4. 重新启动浏览器并进入学习系统

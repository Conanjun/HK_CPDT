  1. First, check to see if you have installed Java Plug-in:

    2. Check to see if the section "Java (Sun)" is found as shown below
    3. If the section is not found, follow step b. below to install Java Plug-in
    4. If the checkbox under "Java (Sun)" is not checked as shown above, please check it and restart your browser. Enter the e-Learning system again to see if you need to update the Java Plug-in
  2. Follow the steps below to install:
    (please note that you need to have administrative permissions of the operating system in order to perform the installation)
    1. Download the Java Plug-in (15.2MB)
    2. Install the Java Plug-in
    3. Restart your browser and enter the e-Learning system again

Update Java Plug-in
If you are using an old version of Java Plug-in, follow the steps below to update:
(please note that you need to have administrative permissions of the operating system in order to perform the installation)
  1. Download the new Java Plug-in (15.2MB)
  2. In Control Panel > Add/Remove Programs, remove the program "Java 2 Runtime Environment" or "J2SE Runtime Environment"
  3. Install the new Java Plug-in
  4. Restart your browser and enter the e-Learning system again
